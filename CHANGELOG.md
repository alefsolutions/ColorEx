# Changelog

## [0.2.0] - 2026-03-03

### Added
- New `colorex` package with modern `Heatmap` API.
- Strategy-based normalization: `linear`, `log`, `quantile`.
- Built-in themes and custom theme support.
- HTML renderer with responsive layout, tile tooltips, and optional legend.
- PNG image renderer using Pillow.
- CLI command: `colorex render input.csv --out output.html`.
- PyPI-ready packaging with `pyproject.toml`.
- Pytest suite covering core contract requirements.
